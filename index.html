This page demonstrates how CSS selectors work.

id

Ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div {
    background-color: yellow;
}

class

Classes can be applied to multiple items on a page

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list {
     color: blue;    
}

compound

A compound selector allows multiple selectors separated by commas. In this way we can apply the same rules to several selectors.

Here's how the CSS is applied to the element:

.new-list, #new-div{
     background-color: black;           
     color: pink;    
}

Here's what it looks like:

  • I'm in an li with class name new-list
I'm inside a div with an id of new-div
child

A child can selector allows us to access the next immediate element(s) inside the current element.

Here's how we hook a child attribute to the parents of an elements:

class="my-kids"

Here's how CSS is applied to the element:

ul.my-kids > li {
        background-color: blueviolet;
}

Here's what it looks like:

  • I'm in an li inside a ul with a class name of my-kids
  • I'm in an li inside a ul with a class name of my-kids
  • I'm in an li inside a ul with a class name of my-kids
descendent

Using a descendent selector allows us to access all elements at any depth within the current element.

Here's how we hook a class attribute to the descendent of elements:

class="grandkids"

Here's how CSS is applied to the element:

div.grandkids li {
          background-color: aqua;
  }

Here's what it looks like:

  • I'm in an li inside a div with a class of grandkids
  • I'm in an li inside a div with a class of grandkids
  • I'm in an li inside a div with a class of grandkids
multiple classes

More than one class can be applied to any item on the page.

Here's how we hook multiple classes to an element:

class="add-border add-background"

Here's how CSS is applied to the element:

.add-border {
        border: 1px dashed #000;
}

.add-background {
        background-color: pink;
}
